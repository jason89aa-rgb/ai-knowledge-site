Step 02. 중급

나만의 AI 직원 채용하기:
API & n8n 자동화

잠자는 동안에도 AI가 일하게 만드는 법

매번 챗봇 사이트에 접속해서 질문하는 건 하수입니다. 중급 단계에서는 AI를 내 업무 흐름(Workflow) 속에 집어넣습니다.

1. API가 뭔가요?

API는 '자판기 구멍'입니다. OpenAI 사이트에 가지 않고, 내 프로그램에서 돈(API Key)을 넣고 결과물을 받아오는 연결 통로입니다. 이것만 알면 엑셀, 노션, 슬랙 어디든 AI를 붙일 수 있습니다.

2. 코딩 없이 자동화: n8n (엔에잇엔)

요즘 가장 뜨거운 노코드 툴입니다. 마우스로 선을 연결해서 프로그램을 짭니다.

🛠️ 실전 예시: 뉴스 자동 요약 봇
  1. (시작) 매일 아침 8시에
  2. (검색) 구글에서 '삼성전자' 뉴스를 검색해서
  3. (AI) ChatGPT에게 3줄 요약을 시키고
  4. (발송) 내 슬랙(Slack)이나 카톡으로 보내줘.

이 과정을 n8n에서는 블록 조립하듯 만들 수 있습니다.

3. 조금 더 욕심낸다면: Python 기초

파이썬을 아주 조금만 배워도 할 수 있는 게 100배 늘어납니다. import openai 한 줄이면 나만의 AI 앱을 만들 수 있습니다.
